实用工具 | 你距离PS大神只差这6款免费在线工具！

发表于 2019-12-22 | 分类于 实用工具 | | 浏览 次

前言

我喜欢摄影，还曾花费过很长时间去潜心观看摄影教学视频。但是，无论如何，总觉得和专业之间存在着很大差距。我很清楚，这不仅是视角和设备的原因，还有很大一部分原因来自于后期处理。

我一直以来是排斥对图片进行后期处理，因为，在我观点里我希望保留最真实、最原始的面貌和回忆。

多年之后我思索发现，并非是不喜欢后期处理，只是自己在后期处理技术上面捉襟见肘，索性给自己找了一个台阶而已。而自己又懒得再去花费那么多时间学习PS等后期处理技术，因为这里面包含太多经验和技术，而且安装臃肿的工具要耗费一些功夫，利用碎片化的时间很难得其精髓。

但是，随着技术的逐渐成熟和丰富，先进的技术越来越多的被应用在图片的处理之中，本文就来介绍6款强大的在线图片处理工具，让图片处理对你来说不再是遥不可及的距离。

remove.bg

https://www.remove.bg/

抠图是图像后期处理中使用频率较高且难度较大的技能，虽然Photoshop、美图秀秀等工具都包含了抠图这项功能，但是这依然具有一定的门槛，非专业人士使用起来不仅速度慢，而且效果较差。

但是，remove.bg的出现将会让抠图问题迎刃而解。它是一款国外的在线抠图神器，只需打开上方链接，把图片上传能够5秒左右把图像完美从背景中抠出。

抠图在技术方面属于图像语义分割领域，目前这方面的工具有很多，例如，爱分割、Face++、Versa等，但是，综合速度、使用便捷度、效果方面还是remove.bg效果较好，前面提到的抠图平台要么需要注册登录、要么付费、要么效果不理想，所以，我个人还是推荐使用remove.bg。

TinyPNG

https://tinypng.com/

当我们报名、注册需要上传照片时，往往很多网站对图片的尺寸有所限制，其中最简单粗暴的方式就是使用windows自带的画图工具进行像素调节，但是这样会图像质量造成很大的影响。

这里要介绍的TinyPNG使用智能有损压缩技术将您的PNG文件的文件大小降低。 通过选择性的减少图片中的颜色，只需要很少的字节数就能保存数据。 对视觉的影响几乎不可见，但是在文件大小上有非常大的差别。

另外，虽然这款工具名称是TinyPNG，但是它不仅仅支持PNG格式图像，还支持JPG格式。此外，它使用非常方便，支持拖拽、上传。

Squoosh

https://squoosh.app/

如果觉得TinyPNG还不能满足你的需求，那可以尝试一下这款Squoosh，它可是大有来头，出自于Google之手，实力自然不可小觑。Squoosh目前支持下列格式图像，

  • OptiPNG
  • MozJPEG
  • WebP
  • Browser PNG
  • Browser JPEG
  • Browser WebP

既然已经推荐了TinyPNG为什么还要推荐一款相同的工具？因为Squoosh有它的独特优势，

  • 支持手动调节图像质量
  • 可视化图像对比
  • 可以安装成桌面应用

我们可以根据自己的需求拖动进度条来调整图像的质量，而且能够清晰可见的看到压缩后图像与原图的对比，另外，Squoosh支持安装到桌面，这样使用起来更加方面。

bigjpg

https://bigjpg.com/

前面提到了图像压缩工具，那么这里就介绍一款图像放大工具。bigjpg是一款基于人工智能的无损图像放大工具，目前支持最大16倍无损放大。

目前PS等工具也支持图片放大，但是，放大的图片后依然有明显的模糊感，边缘的重影以及噪点。bigjpg通过神经网络，针对放大图片的线条、颜色、网点等特点，做特殊的算法调整，所以放大效果非常出色, 色彩保留较好, 图片边缘也不会有毛刺和重影,。更重要的是，影响画质的噪点基本在放大的图片上看不出来。

另外，bigjpg不仅可以在网页端使用，还支持windows、mac、Android桌面应用。

https://www.iloveimg.com/zh-cn/photo-editor

  • 转至JPG

  • HTML转图片

稿定PS

稿定抠图

https://www.gaoding.com/koutu

和前面提到的remove.bg“傻瓜式”抠图不同，稿定抠图是一款更加完善的抠图工具，它不仅支持自动抠图，而且支持通用的标记抠图。

在通用抠图中，我们只需要用画笔标记需要保留的部分、需要去除的部分，这样就可以实现背景的抠除。

在自动抠图中，更加智能一些，它不仅支持人像抠图，还可以自动识别图像中的物体类别，根据图像主题是人像或者商品而自动实现抠图，另外，如果觉得对自动抠图效果不满意，稿定抠图还支持手动修复，这样更加灵活一些。
